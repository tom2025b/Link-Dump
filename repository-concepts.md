## Audit limitations

- A missing README means only that GitHub returned no root README; manifest, language, workflow, and tree evidence were still reviewed.
- Repository size is GitHub's rounded metadata value and is not an emptiness test.
- Dependency inventories are direct manifest observations, not transitive dependency or vulnerability scans.
